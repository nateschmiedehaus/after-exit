# Launch Checklist

## Editorial

- [ ] Name, masthead, and one-sentence thesis locked
- [ ] 3 flagship essays commissioned
- [ ] 10 issue franchises approved
- [ ] contributor pitch guidelines written

## Product

- [ ] public landing page live
- [ ] issue architecture page live
- [ ] operating model page live
- [ ] signup capture wired for founders, sponsors, and contributors

## Partnership

- [ ] PEF relationship memo drafted
- [ ] ownership and governance terms outlined
- [ ] first 20 launch readers identified through PEF

## Commercial

- [ ] sponsor charter written
- [ ] 25-account target list built
- [ ] pitch deck or private sponsor brief prepared

## Events

- [ ] first founder dinner date selected
- [ ] host city chosen
- [ ] discussion prompt and guest mix approved
